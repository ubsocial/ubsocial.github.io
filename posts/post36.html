ID's dos dispositivos

Localizar ID's dos dispositivos
Voltar

Smartphone

  • IMEI: ligar '*#06#'

Desktop

  • Processor id (Linux): cpu get processorid
  • Product serial number tag (PSNT) (Windows): 'wmic bios get serialnumber' ou 'wmic baseboard get product,Manufacturer,version,serialnumber'
    • Linux: sudo dmidecode -t system | grep Serial
    • Linux: sudo dmidecode -s system-serial-number
  • Dispositivos Dell: etiqueta física EST (Embedded Service Tag)
  • CPU ID: Só funciona se ativado (pode ser desativado pelo usuário, sendo desativado por padrão na maioria dos sistemas por motivos de privacidade)
  • Network MAC address: id das placas físicas de rede do dispositivo, mas pode ser alterado pelo usuário
    • Android: configurações - sobre o telefone ou informações do dispositivo - status ou informações de hardware
    • iOS: ajustes - geral - sobre - endereço wi-fi
    • Windows: no CMD, comando 'ipconfig /all' ou 'getmac'
    • Linux: no shell, comando 'ifconfig' ou 'ip addr show' (em eth0 (cabeada) ou wlan0 (wifi) será o HWaddr ou ether)
    • macOS: menu superior esquerdo Apple - preferências do sistema - rede - ethernet ou wi-fi - avançado - aba wi-fi ou hardware
  • HD serial number: pode ser atualizado ou alterado pelo usuário
  • BIOS, Board serial (Linux):
    • hwinfo --bios
    • sudo cat /sys/class/dmi/id/board_serial
  • Root volume serial number: pode ser reformatado e alterado com ferramentas específicas
