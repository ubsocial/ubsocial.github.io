Problemática

Suponha que deseja comprar um smartphone com 256GB de memória RAM. Qual o valor dessa mesma capacidade em MB?



Base decimal

Código binário - Conjunto numérico base decimal = Base 10, ou seja: ^10
2^10 = 1024

Então,
1GB != 1000MB
1GB  = 1024MB

b = bit
B = byte
Se 1B = 8b, então 1B != 1b, portanto: B != b
Ou seja, 1MB != 1Mb, portanto: MB != Mb

Conversões

YB * 1024 = ZB * 1024 = EB * 1024 = PB * 1024 = TB * 1024 = GB * 1024 = MB * 1024 = KB * 1024 = B * 8 = b
(Maior ao menor = *)
(Menor ao maior = /)


Exemplos

4,7GB = ? MB
4,7 * 1024 = 4812,8 MB

4,7GB = ? TB
4,7 / 1024 = 0,004589438 TB


Conclusão

256GB = ? MB
(Incorreto) Se 1GB != 1000MB, então 256 * 1000 = 256.000 MB
(Correto) Se 1GB = 1024MB (1*1024), então 256 * 1024 = 262.144 MB
