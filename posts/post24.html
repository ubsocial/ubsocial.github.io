Habilitar Bluetooth no Debian

Habilitar complemento Bluetooth no Debian
Voltar

Alguns Desktop Environments Debian, como XFCE, não possuem o recurso de Bluetooth habilitado nativamente. Para ativá-lo, basta instalar os complementos do mesmo (O 2º comando destina-se para algumas distros, como o Kali Linux): sudo apt install bluez blueman && sudo systemctl enable bluetooth --now

Algumas distros possuem outros mecanismos gerenciadores de bluetooth. Portanto, o comando acima poderá não ser efetivo para tais casos.
