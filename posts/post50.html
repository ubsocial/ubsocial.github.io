Agent-to-Agent (A2A)

Utilização do A2A Agent em IA
Voltar

Conceito:

A2A (Agent-to-Agent), comunicação entre agentes autônomos. Em Inteligência Artificial distribuída, agentes são entidades de software que atuam de forma autônoma, tomam decisões, comunicam-se com outros agentes e cooperam ou competem para atingir objetivos. A comunicação é central em sistemas multiagentes (MAS). Protocolo utilizado para comunicação é XMPP (Extensible Messaging and Presence Protocol), via XML, onde cada agent possui 'nome@servidor'. Framework SPADE utiliza tal protocolo, onde cada agent conecta-se a servidor, possibilitando enviar e receber mensagens.


Exemplo prático XMPP:

Criação de agents XMPP via framework SPADE, em notebook Python, para comunicação entre agentUser (solicitará dados de local geográfico) e agentMaps (responderá com informações geográficas).

  1. Criar 2 cadastros em https://xmpp.jp:
    • Usuário: agentUser@xmpp.jp - Senha: "abc123"
    • Usuário: agentMaps@xmpp.jp - Senha: "def456"
  2. Obter keys API Google Maps (Geocoding API):
    • Crie conta gratuita no Google Cloud, requer cadastro de cartão de crédito (US$ 200 mensais gratuitos), o exemplo básico não gerará custos. Consulte a lista de preços;
    • No console https://console.cloud.google.com, crie novo projeto;
    • No menu "APIs e serviços", em "Biblioteca", ative Geocoding API;
    • No menu "Credenciais", crie chave de API, e copie-a;
  3. Crie notebook Python 'a2a.ipynb', com conteúdo:

import asyncio
import requests
from spade.agent import Agent
from spade.behaviour import CyclicBehaviour
from spade.message import Message

GOOGLE_API_KEY = "CHAVE_GOOGLE_MAPS"
agent_maps_jid = "agentMaps@xmpp.jp"
agent_maps_pwd = "def456"
agent_user_jid = "agentUser@xmpp.jp"
agent_user_pwd = "abc123"

class AgentMaps(Agent):
    class respostaLocal(CyclicBehaviour):
        async def run(self):
            msg = await self.receive(timeout=15)
            if msg:
                local = msg.body
                print(f"[agentMaps] Local recebido: {local}")
                resposta = self.buscar_maps(local)
                reply = Message(to=str(msg.sender))
                reply.body = resposta
                await self.send(reply)
                print("[agentMaps] Resposta enviada.")
            else:
                print("[agentMaps] Aguardando mensagem")

        def buscar_maps(self, local):
            url = "https://maps.googleapis.com/maps/api/geocode/json"
            params = {"address": local, "key": GOOGLE_API_KEY}
            r = requests.get(url, params=params)
            data = r.json()
            if data["status"] == "OK":
                resultado = data["results"][0]
                nome = resultado["formatted_address"]
                lat = resultado["geometry"]["location"]["lat"]
                lng = resultado["geometry"]["location"]["lng"]
                link = f"https://www.google.com/maps/search/?api=1&query={lat},{lng}"
                return f"Endereço: {nome}\nLatitude: {lat}\nLongitude: {lng}\nLink: {link}"
            else:
                return "Local não encontrado."

    async def setup(self):
        print("[agentMaps] Iniciado")
        self.add_behaviour(self.respostaLocal())

class agentUser(Agent):
    class enviarConsulta(CyclicBehaviour):
        async def run(self):
            print("[agentUser] Enviando consulta...")
            msg = Message(to=agent_maps_jid)
            msg.body = "Cristo Redentor, Rio de Janeiro"
            await self.send(msg)
            resposta = await self.receive(timeout=15)
            if resposta:
                print("[agentUser] Resposta recebida:\n" + resposta.body)
            else:
                print("[agentUser] Nenhuma resposta recebida.")

    async def setup(self):
        print("[agentUser] Iniciado")
        self.add_behaviour(self.enviarConsulta())

maps_agent = AgentMaps(agent_maps_jid, agent_maps_pwd)
user_agent = agentUser(agent_user_jid, agent_user_pwd)

async def main():
    await maps_agent.start(auto_register=True)
    await user_agent.start(auto_register=True)
    await asyncio.sleep(20)
    await user_agent.stop()
    await maps_agent.stop()

asyncio.run(main())
