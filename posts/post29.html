Weka no Linux

Instalação do Weka Machine Learning no Linux
Voltar

Passo a passo:

  1. Pré-requisito (deb): sudo apt install default-jdk
  2. Baixar, na pasta Downloads, Weka oficial (.zip) e extraí-la: Acesse
  3. Permitir execução: sudo chmod +x /home/SEUUSUARIO/Downloads/weka-3-8-6/weka.sh
    • Verifique a versão da pasta gerada para alterá-la nos códigos (Nesse caso foi 'weka-3-8-6')
  4. Criar atalho: sudo nano /usr/share/applications/weka.desktop 
    
Inserir o seguinte conteúdo no arquivo. Ctrl+k para fechar, e salvá-lo
[Desktop Entry]
Encoding=UTF-8
Name=Weka
Comment=Machine learning algorithms for data mining tasks
Exec=/home/SEUUSUARIO/Downloads/weka-3-8-6/weka.sh
Icon=/home/SEUUSUARIO/Downloads/weka-3-8-6/weka.ico
Terminal=false
Type=Application
StartupNotify=false
Categories=Development;IDE;Java;
