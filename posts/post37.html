Astah UML no Linux

Instalar e ativar Astah UML no Linux
Voltar

Baixar Astah UML

  1. Baixar Astah UML ('.deb' ou '.rpm'): https://astah.net/downloads
  2. Instalar Astah UML:
    • Debian: sudo dpkg -i nomeArquivoAstah.deb
      • Em caso de erro de falta de dependências: sudo apt -f install
    • RHEL: sudo dnf install nomeArquivoAstah.rpm

Obter licença estudante

  1. Criar email temporário .edu: https://etempmail.com
  2. Preencher formulário de licença estudante Astah, informando email .edu gerado: https://astah.net/products/free-student-license
  3. Acessar caixa de entrada do email .edu, e baixar licença (pasta .zip contendo arquivo .xml de licença)
  4. Extrair arquivo de licença (.xml) do zip baixado

Baixar JDK específico Adoptium temurin

Instruções conforme site oficial https://adoptium.net/installation/linux

    Debian:
  1. Instalar pré-requisitos: sudo apt install -y wget apt-transport-https gpg
  2. Adicionar chave autenticação: wget -O - https://packages.adoptium.net/artifactory/api/gpg/key/public | sudo apt-key add -
  3. Adicionar repositório: echo "deb https://packages.adoptium.net/artifactory/deb $(awk -F= '/^VERSION_CODENAME/{print$2}' /etc/os-release) main" | sudo tee /etc/apt/sources.list.d/adoptium.list && sudo apt update
  4. Instalar JDK 21 temurin: sudo apt install -y temurin-21-jdk
    RHEL:
  1. Entrar modo administrador: sudo su
  2. Executar comando abaixo (use nome da distribuição abaixo como 'fedora', 'centos' ou 'rhel') 
    
DISTRIBUTION_NAME=fedora
cat <<EOF > /etc/yum.repos.d/adoptium.repo
[Adoptium]
name=Adoptium
baseurl=https://packages.adoptium.net/artifactory/rpm/${DISTRIBUTION_NAME:-$(. /etc/os-release; echo $ID)}/\$releasever/\$basearch
enabled=1
gpgcheck=1
gpgkey=https://packages.adoptium.net/artifactory/api/gpg/key/public
EOF
  3. Atualizar repositórios: sudo dnf update
  4. Instalar JDK 21 temurin: sudo dnf install temurin-21-jdk
  5. Selecionar JDK 21 temurin:
      Debian:
    • sudo update-alternatives --config java # selecionar Java temurin
    • sudo update-alternatives --config javac # selecionar Java temurin
      • RHEL:
    • sudo alternatives --config java
    • sudo alternatives --config javac

Ativar licença estudante Astah UML:

  1. Abrir Astah UML, aba Help > License... > Set License Key > Selecionar arquivo .xml de licença
