Lazarus no Linux

Instalação e configuração do Lazarus IDE no Linux
Voltar
Conteúdo disponível

Instalação:

  1. Baixar Lazarus: acesse
    1. Baixar 'fpc-laz.deb', 'fpc-src.deb' e 'lazarus-project.deb'
  2. Instalar fpc's: sudo dpkg -i fpc-*.deb
  3. Instalar lazarus-project: sudo dpkg -i lazarus-*.deb
    • Se houver erro, instalar dependências: sudo apt -f install
    • Executar novamente comando no tópico 3 acima

Lazarus janela única:

  1. Lista de pacotes disponíveis, procurar 'anchordockingdsgn 1.0' e 'dockedformeditor 0.0';
    • Selecionar pacotes (ctrl+click) e clicar em Instalar Seleção.
  2. Salvar e reconstruir IDE;
  3. Janela 'Editor de código', botão direito, marcar opção 'Travado'.

Lazarus dark mode:

  1. Lazarus segue tema de cores conforme tema GTK do sistema operacional que está instalado;
  2. Para habilitar dark mode no Lazarus, é necessário alterar o tema do sistema operacional para dark mode.
