Formatar pendrive no Linux

Formatar pendrive no terminal Linux
Voltar

Passo a passo:

  1. Listar dispositivos: lsblk
  2. Desmontar pendrive: sudo umount /dev/nomeDispositivo
  3. Formatar pendrive
    1. Formatar FAT32 (comum para compatibilidade com Windows e outros dispositivos): sudo mkfs.vfat /dev/nomeDispositivo
    2. Formatar exFAT (para arquivos maiores que 4GB): sudo mkfs.exfat /dev/nomeDispositivo
    3. Formatar NTFS (comum em sistemas Windows): sudo mkfs.ntfs /dev/nomeDispositivo
    4. Formatar EXT4 (comum em sistemas Linux): sudo mkfs.ext4 /dev/nomeDispositivo
  4. Verificar funcionamento: lsblk && sudo mount /dev/sdX1 /mnt && ls /mnt && sudo umount /mnt
