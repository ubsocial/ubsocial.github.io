Instalação Windows 11 (via Windows, Rufus)

Instalação do Windows 11 oficial (ignorando requisitos de hardware e ativação), em ambiente Windows, via Rufus.

  1. Pré-requisitos:
    • Windows para criação de pendrive bootable
    • Pendrive 8GB
  2. Arquivo ISO (oficial)
    • Windows 11: baixar
    • Windows 10: baixar
  3. Instalar Rufus: acessar
  4. Formatar pendrive (bootable) via Rufus (dados do pendrive serão apagados!)
    • Selecionar device (pendrive)
    • Selecionar ISO
    • Clicar em Start
    • Selecionar "Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0"
    • Selecionar "Remove requirement for an online Microsoft account"
    • Selecionar "Disable data collection (Skip privacy questions)"
    • Clicar em Ok
  5. (instalação em VM) mínimo 3.5GB RAM e 50GB armazenamento
  6. Ao iniciar Windows 11, tela "Informe chave", clicar em "Não tenho a chave do produto"
  7. Selecionar "Windows 11 Home"
  8. Formatar disco e confirmar instalação
  9. (caso solicitar) Tela Windows 11 Adicionar conta, informar e-mail "no@thank.you" e informar senha de preferência
  10. Tela Ops algo deu errado, clicar em Próximo
  11. Informar demais dados e confirmar instalação

Pendrive bootable Windows (via Ubuntu, Gnome Disks)

Criação de pendrive bootable de Windows, em ambiente GNU/Linux (Ubuntu), via Gnome Disks.

  1. Pré-requisito: Pendrive 8GB
  2. Baixar arquivo ISO (links acima)
  3. Abrir Gnome Disks (sudo apt install -y gnome-disk-utility)
    1. Canto esquerdo, clicar no Pendrive para selecioná-lo
    2. Canto superior direito, 3 pontos, clicar em Formatar disco
      • Em Particionamento, selecionar Compatível (GPT)
    3. Clicar em Formatar (dados do pendrive serão apagados!)
    4. Após formatação, clicar em + (criar partição em espaço vazio)
      1. Selecionar todo espaço do pendrive, clicar em Próximo
      2. Informar nome do pendrive (nome do volume) de preferência
      3. Em Tipo, selecionar "Para uso com o Windows (NTFS)"
      4. Clicar em Criar
  4. Em gerenciador de arquivos, na ISO, clicar "Abrir com" e selecionar Montador de imagem de disco (montará disco com arquivos)
    • Copiar todos arquivos desse disco, e colá-los dentro do pendrive
  5. Após finalizar colagem dos arquivos no pendrive, pode-se desplugá-lo, e operação está finalizada

Pendrive bootable Windows (via Linux, Ventoy)

Criação de pendrive bootable de Windows, em ambiente GNU/Linux (geral), via Ventoy.

  1. Pré-requisito: Pendrive 8GB
  2. Baixar arquivo ISO (links acima)
  3. Instalar Ventoy (.tar.gz): acessar
    1. Extraí-lo da pasta
    2. Na pasta extraída, acessar via terminal e executar Ventoy GUI
      1. sudo ./VentoyGUI.x86_64
      2. No Ventoy GUI, em Device, clicar em Refresh
      3. Em Ventoy in Package, pode-se manter MBR (senão pode-se alterar em Options - Partition Style)
      4. Clicar em Install e confirmar formatar pendrive (dados do pendrive serão apagados!)
    3. Após formatação do pendrive, pode-se mover a ISO do Windows para dentro do pendrive, e operação está finalizada (pode-se colocar vários Windows dentro do pendrive - multiboot)
