Habilitar Bash autocomplete no Debian

Habilitar autocompletar no Bash Debian
Voltar

Por padrão, o Debian vem sem o recurso de autocompletar termos no Bash. Para ativar tal recurso, segue abaixo o passo a passo:

  1. Instalar complemento: sudo apt install bash-completion
  2. (Ocasional) Aprimorar arquivo de configuração: sudo nano /etc/profile 
    
Logo no início do arquivo, insira o seguinte trecho:
if [ -f /etc/bash_completion ]; then
 . /etc/bash_completion
fi
