Formatar pendrive via diskpart

Formatação de pendrive pelo diskpart
Voltar

Passo a passo:

  1. Acessar diskpart via CMD Windows: diskpart
  2. Listar discos: list disk
    • Verifique o nº de seu pendrive, conforme tamanho do armazenamento
  3. Selecionar pendrive: select disk numPendrive
    • Pode-se verificar se o pendrive foi selecionado, com list disk
  4. Resetar pendrive: clean
  5. Criar partição: create partition primary
    • Pode-se listar as partições do disco selecionado, com list partition
  6. Formatar partição: format fs=fat32 quick
    • Troque o 'fat32' por 'ntfs' caso opte por formatação ntfs
    • Remova o 'quick' para formatação lenta e completa
  7. Ativar partição: active
  8. Finalizar e criar letra: assign

Outros comandos

  • Selecionar partição de disco selecionado: select partition numParticao
  • Excluir partição selecionada: delete partition
  • Listar volumes: list volume
